# Mobile App

## Instructions

1. Define platform targets, navigation map, state persistence, and offline behavior.
2. Keep device permissions minimal and user-facing.
3. Verify accessibility, performance, crash reporting, and analytics events.
4. Check store policy, privacy labels, and release signing assumptions.
5. Test upgrade, logout, poor network, background, and notification flows.
