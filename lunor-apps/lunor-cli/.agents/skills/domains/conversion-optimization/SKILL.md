# Conversion Optimization

## Instructions

1. Identify the funnel stage, user intent, primary action, and baseline metric.
2. Separate copy, offer, UX friction, trust, performance, and targeting hypotheses.
3. Require event tracking before recommending an experiment.
4. Prefer small measurable changes over broad redesigns.
5. Define success, guardrail metrics, sample constraints, and rollback criteria.
