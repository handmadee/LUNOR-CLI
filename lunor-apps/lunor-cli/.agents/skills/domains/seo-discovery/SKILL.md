# SEO Discovery

## Instructions

1. Map query intent to page type, page depth, and internal links.
2. Check crawlability, canonical policy, structured data, metadata, and indexation risk.
3. For programmatic SEO, require unique value, templates, validation, and duplication controls.
4. For AI search, optimize entity clarity, answer structure, citations, and source trust.
5. Define verification with crawl checks, search console signals, and analytics events.
