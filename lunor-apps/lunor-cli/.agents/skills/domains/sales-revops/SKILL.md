# Sales RevOps

## Instructions

1. Define lifecycle stages and ownership boundaries.
2. Keep routing, scoring, enrichment, and handoff rules explicit.
3. Avoid duplicating CRM source-of-truth fields.
4. Track conversion, velocity, leakage, and data quality metrics.
5. Test edge cases for duplicates, stale owners, missing firmographics, and failed automation.
