# DevOps Deployment

## Instructions

1. Define environment variables and secret sources explicitly.
2. Confirm build, test, migration, deploy, and rollback commands.
3. Require health checks and operational logs for critical paths.
4. Avoid manual-only release steps unless documented.
5. Verify production config does not depend on local development defaults.
