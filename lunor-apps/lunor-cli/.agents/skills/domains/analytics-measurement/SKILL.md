# Analytics Measurement

## Instructions

1. Define the decision the data must support before defining events.
2. Create event names, properties, identity rules, and source-of-truth ownership.
3. Track conversion and guardrail metrics together.
4. Avoid collecting sensitive data unless explicitly required and approved.
5. Add validation checks for duplicate events, missing properties, and identity drift.
