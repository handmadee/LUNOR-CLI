# Frontend UI

## Instructions

1. Define user workflow and component states before implementation.
2. Keep component props explicit and typed.
3. Verify responsive behavior, keyboard access, loading, empty, and error states.
4. Avoid decorative layout that weakens repeated operational workflows.
5. Add focused tests or screenshots for high-risk UI behavior.
