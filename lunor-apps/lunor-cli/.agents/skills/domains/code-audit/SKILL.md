# Code Audit

## Instructions

1. Lead with findings ordered by severity.
2. Ground every finding in file, behavior, and impact.
3. Separate confirmed bugs from plausible risks.
4. Prefer minimal fixes and focused tests.
5. Do not rewrite code unless explicitly asked.
