# Supply Chain Security

## Instructions

1. Inventory package sources, lockfiles, install scripts, and generated artifacts.
2. Flag networked install steps, postinstall hooks, binary blobs, and broad shell permissions.
3. Require pinning or review for third-party skills and plugins.
4. Check CI tokens, release credentials, and package publishing controls.
5. Recommend audit steps without running destructive or untrusted code.
