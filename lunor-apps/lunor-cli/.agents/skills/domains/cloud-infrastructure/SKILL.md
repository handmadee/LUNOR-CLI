# Cloud Infrastructure

## Instructions

1. Identify environments, ownership, credentials, network boundaries, and failure domains.
2. Prefer declarative IaC and repeatable deploys.
3. Check secrets, IAM, logging, backups, and rollback paths.
4. Model cost and scaling assumptions.
5. Verify production readiness with health checks, alarms, and incident runbooks.
