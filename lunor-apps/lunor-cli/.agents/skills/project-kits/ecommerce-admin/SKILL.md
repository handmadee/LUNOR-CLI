# Ecommerce Admin

## Instructions

1. Model admin workflows around repeated operations: search, filter, edit, approve, refund, export, and audit.
2. Apply resource-scoped authorization to every destructive or financial action.
3. Prefer dense, scan-friendly UI over marketing-style layouts.
4. Record audit events for catalog, order, refund, role, and customer-data actions.
5. Add tests for unauthorized access, partial failures, and data export boundaries.
