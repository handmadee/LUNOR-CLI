# Extension Builder

## Instructions

1. Identify the host platform and its permission model before implementation.
2. Isolate host-specific adapters from product/domain logic.
3. Design message contracts and data boundaries explicitly.
4. Include privacy, store-review, licensing, and defensive black-box review before release.
5. Validate extension behavior with small scoped test fixtures.
