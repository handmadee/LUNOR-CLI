# AI Product Lab

## Instructions

1. Define user task, model contract, data inputs, and failure modes.
2. Design evals before optimizing prompts or model choice.
3. Separate research notebooks from production runtime.
4. Add monitoring for quality, latency, cost, drift, and safety failures.
5. Keep sensitive data and prompt logs behind explicit retention policy.
